# jniLibs

## 模块职责

- 放 APK 打包时随应用一同携带的 native libs。
- 这里只放宿主运行期确实需要从 APK 侧带上的本地库。

## 主要公开入口 / 核心类

- 按 ABI 分目录组织的 `.so` 文件

## 上游依赖与下游被谁使用

- 上游由 `app` 模块打包进入 APK。
- 下游由 Android 宿主运行时加载。

## 修改时优先放这里的内容边界

- 宿主运行期必须随 APK 提供的本地库。
- 与 ABI 打包直接相关的 native 产物。

## 不该放进来的内容

- 不会随 APK 发布的调试产物。
- 应该放在 bootstrap/rootfs 里的运行时文件。
- 与 native library 打包无关的任意二进制文件。
