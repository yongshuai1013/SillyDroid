# core/common

## 模块职责

- 放纯基础公共能力。
- 当前主要是 dispatcher 这类不带业务语义的共通基础件。

## 主要公开入口 / 核心类

- `Dispatchers.kt`

## 上游依赖与下游被谁使用

- 作为底层基础模块，被 `domain` 直接依赖。
- 再由 `domain` 向 `data`、`feature`、`ui` 传递。

## 修改时优先放这里的内容边界

- 与业务无关、但需要跨模块复用的极小基础能力。
- 调度器、线程切换、通用基础接口这类内容。

## 不该放进来的内容

- Android 页面能力。
- bootstrap、下载、更新、日志这类带业务语义的模型或实现。
- 任何“为了方便”塞进来的工具杂项。
